LibreHardwareMonitor instellen voor energiemeting (Thesis §5.4 — verplicht voor actieve metingen)
  1. Download LibreHardwareMonitor van GitHub en pak het zip-bestand uit
  2. Start LibreHardwareMonitor.exe als Administrator
  3. Activeer de webserver: Options → Remote Web Server → Run (poort 8085)
  4. Verifieer: http://localhost:8085/data.json moet een JSON-boom teruggeven met minstens CPU- en GPU-power-sensoren

Tijdens benchmarks kan de runner de totale gemeten energie en componenten automatisch invullen via LibreHardwareMonitor. Warm-up runs worden niet opgeslagen in de DB.
